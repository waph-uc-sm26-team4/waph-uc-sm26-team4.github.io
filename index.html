Planned Features

  • User registration with email as username and password
  • User login and logout
  • Password change
  • Profile editing, including name, additional email, and phone
  • Creating, editing, and deleting posts
  • Viewing and adding comments on posts
  • Superuser account management features
  • Secure coding protections against XSS, SQL injection, CSRF, and session attacks
